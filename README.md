# portfolio

##  ダサいポートフォリオサイト

## 必須項目

- カウンター
  - 何とかして実装
- ニュース(更新履歴でスクロールできるやつ)
  - コミットログから表示する仕組みを作る
- ダサいメニュー画像
  - 適当に自作する
- リンクはトップページ意外禁止
  - テキストで貼る
- MIDI で音楽が鳴る
  - めんどそう
- マウスカーソルに星屑が付く
  - がんばればできそう
- ページの切り替えで派手なエフェクト
  - がんばる

## 入れたい古代機能

- 背景画像は repeat させたもの
- 古代モードのとき marquee タグでタイトルを動かす
- ダサい戻るボタン
- スマホから古代モードで見たときガラケー風の表示
